About Kronos Bot

Work smarter by using Kronos Bot to quickly get information from Workforce Central application. Ask questions about your vacation balance. See who is approaching overtime on your team this week. Find out what your schedule is for next week and much more...

The main goals of the solution are:

  • Simplify labor management by asking questions to Kronos Bot rather than logging into the web application.
  • Improve user experience and productivity by quickly providing important schedule and time and attendance information with ease.
Common Information

Kronos Bot consists of fifteen dialogues. Five of them (Overtimes, Where is Someone Today, Who is Here, Who is Not Here, My Employee's Time Off Requests) are available only for managers.

Authentication

Kronos WFC Bot currently only supports authentication with your Kronos credentials using Click to Log In option.

Summary

As of now Kronos Bot supports a limited number of dialogues, mostly related to the Timekeeping and Accruals. The Bot is able to recognize spoken or text input. It is trained to understand commonly used phrases, ask additional questions if needed, and provide answers.

    Kronos Bot supports the following dialogues:

    • Accrual Balance
    • Add Punch
    • Hours Worked
    • List of Punches
    • My Upcoming Shifts
    • My Time Off Requests

    • My Schedule
    • Sick Day

    • Create Swap Shift Request
    • Create Time Off Request

    Accrual Balance

    This dialogue will display accrual balances for the user.

      Dialogue Examples:

      • How many vacation hours do I have?
      • Show my accruals
      • Give me my vacation balance
      • Accrual balance
      • My available vacation hours
      • What’s my vacation balance?
      Add Punch

      This dialogue helps user to add punch or punch in with transfer.

        Dialogue Examples:

        • Add Punch
        • Add Punch with Transfer
        • Punch in with transfer
        • Clock in with transfer
        • Transfer me to
        Hours Worked

        This dialogue will display the number of hours the user worked during the specified period.

          Dialogue Examples:

          • Show me hours I worked this week
          • Give me my hours worked today
          • Number of hours I worked from Monday till Friday
          • How many hours did I work this month?
          • My worked hours
          List of Punches

          This dialogue displays the user`s punches for the given period.

            Dialogue Examples:

            • Show me my punch list
            • Give me a list of punches for today
            • I need to get punches for yesterday
            • I want to see punches from the 1st of September till today
            • Can you show punches created this Monday?
            • Today's list of punches
            My Upcoming Shifts

            Dialogue will return shifts with transfer (if exist) for one specified day.

              Dialogue Examples:

              • What are my shifts for Tuesday?
              • What are my transfers for Tuesday?
              • When is my next shift?
              • Shifts next Friday
              • My shifts for tomorrow
              • Show shifts for Tuesday
              • Show me my shifts for the 1st of February
              • Show my upcoming shifts
              My Time Off Requests

              This dialogue will display all time off requests from today.

                Dialogue Examples:

                • When is my next vacation?
                • Show me my time off requests
                • My time off requests
                • Has my paid time off been approved?
                • Has my PTO been approved?
                • Has my time off been approved?
                • Has my leave been approved?
                • Has my vacation been approved?
                • Show vacation requests
                • Next vacation requests
                Overtimes

                This dialogue (available only to managers) shows the number of overtime hours for the team members.

                  Dialogue Examples:

                  • Who is approaching overtime this week?
                  • Show me the number of overtime hours for current pay period
                  • Give me the team's overtimes from September 4 to October 10
                  • Number of overtime hours for the team last month
                  • Overtime hours for this Monday
                  • Today's employee overtimes

                  NOTE: This dialogue is available only for managers.

                      • Next Sunday schedule

                      • Show me last week’s schedule

                      Sick Day

                      This dialogue helps the user to submit a sick day off. Internally it creates a "Sick Hours" pay code edit with the specified duration for the given date.

                        Dialogue Examples:

                        • I would like to take 4 hours sick day off for today
                        • 8 hours sick leave
                        • I need to take sick leave
                        • I want to have a sick day off
                        • Can I have a sick day off for tomorrow?
                        • I need 8 hours sick leave on the 3rd of November
                        • Give me sick leave for next Monday
                        Where is Someone Today

                        This dialogue (available only to managers) displays a list of employees` locations for today.

                          Dialogue Examples:

                          • Where is Eliza Adams today?
                          • Eliza Adams location
                          • Eliza location
                          • Adams location today
                          • Eliza Adams location with punches
                          • Show me where is Eliza

                          NOTE: This dialogue is available only for managers.

                            Who is not here

                            This dialogue displays a list of employees (of the logged in manager) who did not clock in today.

                              Dialogue Examples:

                              • Missing in punches for the team
                              • Who is not here?
                              • Who didn't clock in today?
                              • Who didn't add in punches?
                              • Who didn't punch in today yet?
                              • Who is not here from my team?
                              • Absence list

                              NOTE: This dialogue is available only for managers.

                                Who is here

                                This dialogue displays a list of employees (of the logged in manager) who clocked in today.

                                  Dialogue Examples:

                                  • Who is here
                                  • Who clocked in today?
                                  • Who punched in today?
                                  • Who is here from my team?
                                  • Presence list

                                  NOTE: This dialogue is available only for managers.

                                    Swap Shift Request

                                    This dialog allows an employee to define and submit a request to swap a shift with a colleague. The dialog is available for Microsoft Teams only.

                                      Dialogue Examples:

                                      • Swap shift
                                      • Swap shift for tomorrow
                                      • Create swap shift request for next monday
                                      Time Off Request

                                      This dialog allows an employee to define and submit a request to be excused from regular work duties for different reasons. The dialog is available for Microsoft Teams only.

                                        Dialogue Examples:

                                        • Submit time off request
                                        • I need to take time off for next monday
                                        • Create vacation request for tomorrow
                                        • Take vacation for the 1st, October, 2018
                                        My Employee's Time Off Requests

                                        This dialog allows manager to find time off requests created by her employees. The dialog is available for Microsoft Teams only.

                                          Dialogue Examples:

                                          • Time Off requests of employees
                                          • Eliza Adams time off requests
                                          • Employee's leave requests for today
                                          • My employee's vacation requests next month
